### 🛠 现代全栈架构 | Tech Stack

#### 📦 Infrastructure & Engineering
- **Workflow:** Monorepo managed by **Turborepo** & **NPM Workspaces**.
- **Edge Computing:** Heavy use of **Cloudflare Ecosystem** (Workers, Pages, KV, R2, Hyperdrive).
